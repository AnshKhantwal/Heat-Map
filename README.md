Description
-> A single-page HTML application with inline JavaScript that visualizes patient distribution on an interactive map. The dashboard fetches live data from Google Sheets and displays patient density using a Leaflet-based heatmap with branch-wise filtering.

Features
-> Single HTML file with inline CSS and JavaScript
-> Interactive heatmap using Leaflet.js
-> Branch, month range, and year filtering
-> Live data from Google Sheets (CSV export)
-> Branch location markers
-> Fuzzy matching for area names
-> Optional Google Geocoding API support
-> Automatic zoom and patient count display

Technologies Used
-> HTML5, CSS3
-> JavaScript (inline scripting)
-> Leaflet.js & Leaflet.heat
-> OpenStreetMap
-> Google Sheets (CSV)

-> index.html
